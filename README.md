# Hospital Management System (C++)

A console-based Hospital Management System built with C++ that helps manage patients, doctors, and appointments using file storage for persistent data.

This project demonstrates Object-Oriented Programming, file handling, and data management using C++.

# Project Overview

The Hospital Management System is a simple application designed to simulate hospital record management.

It allows users to:

* Register new patients
* Add doctors with specialties
* Book appointments between patients and doctors
* View stored hospital records

All data is stored in text files, allowing the system to preserve information even after the program closes.

# Features

Add new patients
View patient records
Add doctors with specialties
View doctor records
Book appointments
View scheduled appointments
Persistent data storage using text files
